## 2. Description of Service

### 2.1 What is NeedMCP

NeedMCP is a specialized MCP (Model Context Protocol) server that acts as a bridge between your AI assistant (such as Cursor or Claude) and a curated library of themed UI components.

### 2.2 Key Features

- **Theme Selection** - Select and lock specific design themes (Glassmorphism, Elegant, Industrial, etc.)
- **Universal Components** - Themed UI components based on Design Token
- **Cross-Framework** - Components can be adapted to React, Vue, Angular, Svelte, Flutter, etc.
- **Design System** - Complete documentation on colors, typography, and component guidelines

### 2.3 Availability

We endeavor to keep the service available 24/7, but we do not guarantee 100% uptime. We reserve the right to perform scheduled maintenance with or without prior notice.
