# needmcp-cli

NeedMCP official CLI — Setup MCP configuration for AI coding assistants.

## Installation

```bash
npm install -g needmcp
```

Or use directly with npx:

```bash
npx needmcp --help
```

## Usage

```bash
# Interactive setup
needmcp setup

# Setup with API key
needmcp setup --key sk_need_xxxxxxxx

# Remove NeedMCP from clients
needmcp remove
```
